## 六、ue_bridge.py 可调参数

文件位置：`~/rosCar/ue_bridge.py`

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `MAX_LINEAR_SPEED` | `1.0` m/s | speed=100 对应的最大速度 |
| `MAX_ANGULAR_SPEED` | `1.0` rad/s | 最大转向角速度 |
| `ARRIVE_THRESHOLD_M` | `0.5` m | 到达判定距离，GPS精度不足时调大 |
| `HEADING_KP` | `1.2` | 转向比例系数，实车测试后调整 |
| `HEADING_TOLERANCE` | `0.15` rad | 朝向误差容忍（约8.6°） |

---

## 七、坐标导航工作原理

1. 收到目标经纬度后，计算当前位置到目标的方位角
2. 原地转向，直到朝向误差 < `HEADING_TOLERANCE`
3. 直行，每个控制周期（100ms）重新计算方位修正
4. 距目标 < `ARRIVE_THRESHOLD_M` 时停车，向 UE 回复 `arrived`
5. 新指令到来时自动取消当前导航任务

> 后续接入 Nav2 后，步骤2~4可替换为发送 `NavigateToPose` action，步骤1保留。
